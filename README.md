Picture.ly is an image twitter search tool

Create .env with the following

TWITTER_KEY = your_key  
TWITTER_SECRET = your_secret  
TWITTER_ACCESS_KEY = your_token  
TWITTER_ACCESS_SECRET = your_token_secret
